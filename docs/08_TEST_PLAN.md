# TEST PLAN

## 自動テスト

自動テストでは最低限、以下を検証する。

- 国数が196である。
- difficulty件数が30 / 50 / 56 / 60である。
- 表示名の重複がない。
- `id`の重複がない。
- `flagFile`の重複がない。
- 各`flagFile`が存在する。
- `reading`に空欄がない。
- 首都に空欄がない。
- 人口基準年がある。
- 面積がある。
- `trivia`の重複を検出する。
- ヒントに国名が含まれていない。
- 3択に重複がない。
- 正解が1つだけである。
- 各難易度で問題を生成できる。
- 1000問以上継続できる。
- CPU難易度ごとの正解率が設定どおりである。
- ライフ0で終了する。
- `localStorage`へ保存できる。

## 手動確認項目

- 3択が横並びである。
- 国旗比率が正しい。
- 余白が大きすぎない。
- ヒントが段階式である。
- 回答後の学習情報が読みやすい。
- 解説時間設定が動く。
- 「もっと見る」表示中は自動で進まない。
- CPUターン中に人間が押せない。
- メニューから安全に戻れる。
- JavaScriptエラーがない。

## リリース判定

次の場合はリリース不可とする。

- チェックリスト未完了
- テスト未実行
- 既知の重大不具合あり
- 仮データあり
- スマホ確認なし
- 国旗不正確
- 難易度件数不一致
- 行き止まり発生
