# GAME SPEC

## しりとりルール

- 現在の国名の最後の文字から始まる国を選ぶ。
- 濁点、半濁点は外して判定する。
- 長音「ー」は無視する。
- 小さい文字は通常文字へ変換する。
- 中黒、空白、記号は判定時に無視する。
- 「ん」で終わる国は正解候補に出さない。
- 不正解時も正解国からしりとりを続ける。

## 1人モード

- ライフは3個。
- 不正解でライフを1個減らす。
- ライフ0で終了する。
- スコアは累計正解数とする。
- 現在の連続正解数を記録する。
- 最高連続正解数を記録する。
- 自己ベストを記録する。
- 自己ベストは`localStorage`へ保存する。
- 10問固定ではなく、3回間違えるまで続くエンドレス形式とする。

## 行き止まり回避

- 未使用国を優先する。
- 候補不足時は使用回数1回の国を許可する。
- さらに不足した場合は使用回数上限を段階的に広げる。
- 現在の国は直後に再使用しない。
- 直前の国も可能な限り避ける。
- 候補不足だけでゲームを終了しない。

## 難易度

- 初級：difficulty 1、30か国。
- 中級：difficulty 1〜2、80か国。
- 上級：difficulty 1〜3、136か国。
- 激ムズ：difficulty 1〜4、196か国。
- 上位難易度は下位難易度を含む累積式とする。
- 候補不足時のみ上位難易度を補助候補として使用できる。

## ローカル対戦

- 2〜4人で遊ぶ。
- 同じ端末を順番に使用する。
- 各プレーヤーのライフは3個とする。
- ライフ0で脱落する。
- 最後まで残った人が勝つ。

## CPU対戦

- プレーヤーとCPUが交互に回答する。
- 両者のライフは3個とする。
- 出題難易度とCPU強さは別々に設定する。
- CPU強さは初級、中級、上級、激ムズの4段階とする。
- CPUターン中は人間の操作を無効化する。

## ヒント

- 国名を直接表示しない。
- 地域、首都、食べ物、観光地、文化、言語、隣国、国旗の特徴などを段階式で表示する。
- 次の問題ではリセットする。
- CPUはヒントを使用しない。

## 回答後の学習情報

- 正解国の国名
- 首都
- 人口
- 人口基準年
- 面積
- 言語
- 通貨
- 食べ物
- 観光地
- 名産
- 国旗の意味
- 固有の豆知識
- 短い表示と「もっと見る」の詳細表示を分ける。

## 回答後表示時間

- 「短い」「ふつう」「長い」「自分で進む」から選択できる。
- 設定を`localStorage`へ保存する。
- 「もっと見る」表示中は自動遷移を停止する。
- メニューへ戻るとタイマーを解除する。
