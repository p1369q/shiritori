## Flag artwork

The flag assets in `flags/*.svg` use Unicode regional-indicator flag emoji glyphs for each country's ISO 3166-1 alpha-2 code. Rendering depends on the user's installed color emoji font (for example, Apple Color Emoji, Segoe UI Emoji, or Noto Color Emoji).
